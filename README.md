<h1>Веб api для сравнения коллекций в C#</h1>
<h3>Сравнение происходит по скорости добавления, скорости поиска, скорости удаления и потреблению памяти. Скорость вычисляется в секундах, память в МБ</h3>
<h4>Список эндпоинтов:</h4>
<ul>
  <li>GET: api/benchmark/types - получить коллекции, которые можно добавить в сравнение</li>
  <li>POST: api/benchmark/addcollection?type={тип коллекции} - добавить коллекцию в сравнение</li>
  <li>POST: api/benchmark/count?count={число элементов} - количество элементов в каждой коллекции</li>
  <li>GET: api/benchmark/selectedtypes - получить коллекции, которые уже добавлены в сравнение</li>
  <li>GET: api/benchmark/add - сравнить выбранные коллекции по добавлению элементов</li>
  <li>GET: api/benchmark/search - сравнить выбранные коллекции поиску элемента</li>
  <li>GET: api/benchmark/memory - сравнить выбранные коллекции по потреблении элементов</li>
  <li>GET: api/benchmark/remove - сравнить выбранные коллекции по удалению элементов</li>
</ul>
